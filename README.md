# my-awesome-ai

Централизованный репозиторий для AI-промптов, шаблонов и заметок в формате Markdown (Obsidian-friendly).

## 📁 Структура

- `notebooklm/`
  - `books/` — промпты и разборы по книгам
    - `eric-berne_games-people.md`
  - `youtube/` — шаблоны и примеры конспектов по видео
    - `average-video/summary.md`
    - `tech-video/` (`prompt.md`, `summary-template.md`, `summary-check-list.md`, схемы и примеры)
- `productivity/`
  - `day-planning/` — промпты для планирования дня
    - `prompt.md`
- `templates/` — переиспользуемые мета-шаблоны для сборки новых промптов
  - `prompt-builders/`
    - `prompt-architect-v1.md` — конструктор промптов по 6 блокам (Task/Context/Exemplars/Persona/Format/Tone)
  - `language-learning/`
    - `book-club-esl-companion.md` — интерактивный промпт для разбора английской художественной книги (B1-B2, без спойлеров)
- `tips/` — короткие практические заметки и правила работы с AI
  - `ai-usage.md` — место для персональных best practices

## 🚀 Как использовать

1. Выбери домен задачи: `notebooklm` или `productivity`.
2. Возьми за основу подходящий промпт/шаблон.
3. Для сборки нового промпта используй `templates/prompt-builders/prompt-architect-v1.md`.
4. Фиксируй рабочие приемы в `tips/ai-usage.md`.
