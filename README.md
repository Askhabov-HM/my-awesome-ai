# my-awesome-ai

Централизованный репозиторий для AI-промптов, шаблонов и заметок в формате Markdown (Obsidian-friendly).

## 📁 Структура

- `notebooklm/`
  - `books/` — промпты и разборы по книгам
    - `eric-berne_games-people.md`
  - `youtube/` — формат и заметки по видео
    - `summary.md`
- `productivity/`
  - `day-planning/` — промпты для планирования дня
    - `prompt.md`
- `templates/`
  - `prompt-skeleton.md` — базовый каркас промпта
  - `output-formats/`
    - `obsidian-note.md`
    - `youtube-summary.md`
- `tips/`
  - `ai-usage.md`

## 🚀 Как использовать

1. Выбери нужный раздел (`books`, `youtube`, `productivity`).
2. Скопируй шаблон/промпт.
3. Адаптируй под задачу.
4. Сохрани результат в соответствующую папку.
